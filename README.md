# 🎓 Python CGPA Calculator

A lightweight, interactive command-line tool built in Python to quickly calculate your semester GPA (Grade Point Average) or CGPA. It takes your raw marks, converts them into a standard letter grade, and generates a clean tabular scorecard.

## ✨ Features
* **Dynamic Inputs:** Choose exactly how many subjects you want to calculate for the current semester.
* **Standardized Grading:** Automatically converts raw marks (0-100) into standard letter grades (O, A+, A, B+, B, C, F) and their corresponding grade points.
* **Weighted Calculation:** Accurately calculates your final GPA based on the specific credit weight of each subject.
* **Clean Tabular Output:** Utilizes the `pandas` library to print a beautifully formatted scorecard directly in your terminal before revealing your final GPA.
